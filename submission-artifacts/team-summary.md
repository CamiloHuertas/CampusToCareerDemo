# Team Collaboration Summary

## Project

CampusToCareer is a verified platform connecting current university students with alumni mentors and employers for paid mentorship sessions and short-term paid micro-internships.

## Collaboration Approach

The team divided work by system responsibility. Analysis work focused on actors, use cases, and domain concepts. Design work converted those findings into diagrams, data structure, and interface screens. Prototype work focused on a professional React MVP that demonstrates the main workflows clearly enough for presentation.

## Contributions

| Team Member | Contribution |
|---|---|
| Student 1 | Business scenario, use cases, actor descriptions, and presentation organization |
| Student 2 | Domain model, database schema, architecture/deployment diagrams, and test plan |
| Student 3 | React prototype implementation, UI styling, workflow mock-ups, and final packaging |

## Prototype Scope

The MVP demonstrates the user-facing behavior of the system rather than a production backend. Payment processing, `.edu` verification, meeting links, file uploads, and notifications are represented through mock UI states and diagrams so the presentation can explain how the complete system would work.

## Final Deliverables

- React source code for the CampusToCareer MVP
- Updated UML/system diagrams
- Database schema
- System test plan
- Presentation slide content
- Collaboration summary
