# CampusToCareer Prototype

CampusToCareer is a React MVP for a verified career marketplace where students book paid mentorship sessions with alumni and apply to short-term paid micro-internships posted by employers.

## What Is Implemented

- `.edu` verification status display
- Student dashboard with mentor search
- Mentorship booking quote with 10% platform fee
- Micro-internship marketplace cards
- Employer project posting form and status workflow
- Escrow/payment ledger mock-up
- Trust and safety dispute queue
- Responsive professional UI for desktop and mobile presentation

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Submission Artifacts

The `submission-artifacts` folder includes:

- `system-models.md`: use case diagram, domain model, sequence diagram, architecture diagram, and deployment diagram
- `database-schema.sql`: actual relational schema for the MVP
- `test-plan.md`: system test plan mapped to major use cases
- `presentation-slides.md`: slide-by-slide presentation script/content
- `team-summary.md`: brief collaboration and contribution summary

## Notes

This is intentionally an MVP/prototype. Payment processing, university verification, video links, and notifications are represented as realistic UI flows and architecture components, not production integrations.
